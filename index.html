☁️ Com o Supabase os dados ficam guardados na nuvem e persistem em qualquer computador. Gratuito até 500MB. Siga o guia abaixo para configurar.

📋 Como configurar (5 minutos):

1. Acede a supabase.com e cria uma conta gratuita

2. Clica em New Project → dá um nome (ex: portal-simulador) → escolhe uma senha forte → clica Create

3. Aguarda ~2 minutos até o projecto estar pronto

4. No menu lateral vai a SQL Editor e executa este comando:

CREATE TABLE contribuintes (
  id bigint generated always as identity primary key,
  tipo text,
  nome text not null,
  nif text not null unique,
  pass text not null,
  cae text,
  forma text,
  grupo text,
  municipio text,
  divida numeric default 0,
  creditos numeric default 0,
  criado_em timestamptz default now()
);

-- Permitir acesso público (apenas leitura/escrita via anon key)
ALTER TABLE contribuintes ENABLE ROW LEVEL SECURITY;
CREATE POLICY "acesso_publico" ON contribuintes FOR ALL USING (true) WITH CHECK (true);

5. Vai a Settings → API e copia:

• Project URL → cola no campo Supabase URL acima

• anon / public key → cola no campo Anon / Public Key acima

6. Clica Guardar Configuração e depois Testar Ligação ✅
